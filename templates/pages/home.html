CRUD - Investigacion

Aplicación web en Flask que consume una API genérica en C# para gestionar las tablas del sistema de Mapa de Conocimiento, incluyendo áreas, líneas de investigación y sus relaciones.

Tablas disponibles: Área de Conocimiento, Área de Aplicación, Línea de Investigación, ODS, Relaciones: AC-Línea, AA-Línea, ODS-Línea, Semillero, Semillero-Línea, Grupo de Investigación.
Use el menú lateral para navegar y gestionar cada entidad del sistema.
